사용자 더미데이터 왕창 만들기 (userData.json)

사용자 더미데이터를 이용하여-
로그인 페이지에서 로그인시, 더미데이터에 있는 데이터와 일치하면 로그인 성공

로그인 실패시 아이디 혹은 비밀번호가 틀리면 
"아이디 **혹은** 비밀번호가 틀렸습니다."라고 출력하기.

더미데이터에 아예 존재하지 않는 userName과 userPassword라면
"존재하지 않는 회원입니다." 라고 출력하기.

회원가입 페이지에서 중복확인시, 더미데이터에 userEmail중 이미 존재하는
userEmail이면 오류를 출력하고, **회원가입이 불가하게 만들기**






로그인 페이지에서 뒤로가기 버튼을 누르면 **메인화면**으로 이동하기

로그인 페이지에서 로그인 버튼을 누르고 **성공하면** 맵 화면으로 이동하기

회원가입이 **완료되면** 맵 화면으로 이동하기





해야됨
로그인 또는 회원가입이 완료되면 더미데이터의 isLogin를 true로 바꿔주고
isLogin이 false이면 맵화면으로 이동이 안되게하기
